# Lottie JSON Directory

外包交付 Lottie JSON 后，将以下 13 个文件放入本目录：

## 主状态（6 个）
- `enter.json`
- `idle.json`
- `listen.json`
- `thinking.json`
- `narrating.json`
- `done.json`

## 过渡片段（7 个）
- `idle_to_listen.json`
- `idle_to_thinking.json`
- `listen_to_thinking.json`
- `thinking_to_narrating.json`
- `narrating_to_done.json`
- `thinking_to_listen.json`
- `narrating_to_listen.json`
